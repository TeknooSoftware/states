\Teknoo\States\Loader

\Teknoo\States\LoaderLoaderComposer

Class LoaderStandard Default implementation of the "stated class autoloader".

It is used to allow php to load automatically stated classes without a specific behavior from the developer. It builds on the Composer Loader. It is registered to be called before the composer loader, find the factory attached to the stated class, load and run it to initialize the stated class.
