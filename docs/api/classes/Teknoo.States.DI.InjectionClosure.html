\Teknoo\States\DI

\Teknoo\States\DIInjectionClosure

Class InjectionClosure Default Injection Closure container to use to extract and manipulate all methods of a stated class in the proxy. These containers implement also a "static" mechanism to allow developers to use clean static var in these functions.
