\UniAlteri\States\Proxy

\UniAlteri\States\ProxyStandard

Class Standard Default Proxy class to use in stated class when no proxy are defined in these classes.

A stated object is a proxy, configured for its stated class, with its different stated objects. It is a proxy because, by default, all calls are redirected to enabled states. $this in all methods of the stated object (also of states' methods) points the proxy object.

The library creates an alias with the proxy class name and this default proxy to simulate a dedicated proxy to this class.
